# Генератор данеток — презентация

Интерактивная B2B-презентация автоматизированного SaaS-конвейера для генерации данеток из реальных историй. Шесть слайдов с навигацией с клавиатуры и демо-симулятором пайплайна на мок-данных.

## Содержание презентации

1. **Cover** — обложка и описание продукта
2. **Problem** — проблемы рынка контент-мейкеров
3. **Conveyor** — интерактивная симуляция генерации данетки
4. **Tech** — микросервисная архитектура (Parser, Kafka, AI Worker, Puzzle API)
5. **Metrics** — бизнес- и технические метрики успеха
6. **Win-Win** — unit-экономика и roadmap

## Требования

- Node.js 18 или новее
- npm

## Запуск локально

```bash
npm install
npm run dev
```

Приложение откроется на [http://localhost:3000](http://localhost:3000).

## Сборка

```bash
npm run build
npm run preview
```

## Управление

- Стрелки влево / вправо — переключение слайдов
- Клавиши 1–6 — переход к конкретному слайду
- На слайде «Conveyor» — кнопка «Запустить тест-генерацию» для демо-симуляции

## Стек

- React 19
- TypeScript
- Vite
- Tailwind CSS 4
- Motion (анимации слайдов)
